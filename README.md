# 🎮 俄罗斯方块 Tetris

一个用纯 HTML/CSS/JavaScript 实现的经典俄罗斯方块游戏。

## ✨ 特性

- 🎨 现代化渐变 UI 设计
- 🎯 经典俄罗斯方块玩法
- 📊 分数、等级、行数统计
- 👀 下一个方块预览
- ⏸️ 暂停功能
- 🎮 流畅的动画效果

## 🚀 玩法

| 按键 | 功能 |
|------|------|
| ← → | 左右移动 |
| ↓ | 加速下落 |
| ↑ | 旋转方块 |
| 空格 | 快速落地 |
| P | 暂停/继续 |

## 🏗️ 运行方式

直接用浏览器打开 `index.html` 文件即可！

```bash
# 或者用本地服务器
cd tetris
npx serve .
```

## 📦 部署

已配置 GitHub Pages，访问 https://你的用户名.github.io/tetris/
